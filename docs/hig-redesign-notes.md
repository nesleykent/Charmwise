## Key improvements

- Purpose: The recommendations page now asks the user's real question directly and puts the final pick before comparison, creature summaries, upgrades, and audit details.
- Agency: Manual charm selection, removable chips, segmented views, and collapsed advanced sections keep the user in control without forcing a long generated ranking.
- Responsibility: The local-data privacy note remains visible but quiet, and formula/debug details stay available for transparency without crowding the default answer.
- Familiarity: Navigation, segmented controls, buttons, disclosure rows, fields, and tables now use consistent native-app geometry and system typography.
- Flexibility: Shared page shells and responsive navigation preserve context across desktop and mobile while keeping core controls reachable.
- Simplicity: The old decorative gradient header, serif headline treatment, and repeated frosted-card look were replaced with calmer hierarchy and fewer competing surfaces.
- Craft: Shared tokens now define graphite surfaces, Apple-style blue selection, semantic green/amber/red states, focus rings, borders, and shadows consistently.
- Delight: Visual polish comes from clear feedback, subtle depth, and confident spacing rather than decorative artwork or unrelated effects.
